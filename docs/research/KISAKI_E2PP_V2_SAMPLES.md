### 关键发现

1. **RAG 幻觉**：`kisaki_rag_010` 在知识库仅提供 2 个因果推导口癖的情况下，自行扩展到 10 个词。尽管 citation_hits 命中正确，但内容超出了文档范围。建议在 RAG prompt 中强化「仅使用文档中出现的词」约束。

2. **引用格式不一致**：`kisaki_rag_013` 出现了 `[文档ID:kisaki_world]` 格式，系统未识别该引用（citation_ratio=0.5）。怀疑是特殊 token 编码问题或模型格式不稳定。

3. **多轮对话格式问题**：multiturn 样本的 prompt 以 `|` 分隔多轮，但部分样本（003、015）的模型输出似乎忽略了多轮上下文，仅对最后一轮做单轮回答，导致角色漂移。建议检查 `|` 格式是否正确注入 system prompt。

4. **角色自指问题**：部分 persona 样本（008、010、022）出现「我是一个AI」「我是语言模型」等自指，严重破坏了角色扮演体验。E2'' 重训后此问题依然存在，建议增加「禁止自称为 AI/模型」的 system prompt 约束。
